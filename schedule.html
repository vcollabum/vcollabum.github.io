Book Schedules

Berikut ini merupakan prosedur penggunaan Virtual Machine(VM) Classroom agar kegiatan praktikum dapat berjalan lebih efektif, antara lain:

  1. Jadwal VM Classroom akan dibersihkan dalam satu minggu sekali.
  2. Sebelum menggunakan VM, mahasiswa dipastikan untuk mendapaftar jadwal minimal satu minggu sebelum jadwal praktikum
  3. Students are required to fill in their name, email, and contact number as coordination regarding the use of VM Ware.
  4. Make sure to select an empty schedule. Every day there are 3 different hours with code 1: 10:00AM - 12:00PM; code 2: 1:00PM - 3:00PM; and code 3: 3:00PM - 5:00PM
  5. After the student has registered, the student will be given access by the admin via email.
  6. If there is a problem, please contact the admin immediately.
